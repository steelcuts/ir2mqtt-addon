# IR2MQTT — Home Assistant Add-on Repository

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fsteelcuts%2Fir2mqtt-addon)

This is the official Home Assistant Add-on repository for **IR2MQTT** — a Web UI for IR bridges with device learning, IR code databases, and MQTT Auto-Discovery for Home Assistant.

---

## Installation

1. Click the button above, or go to **Settings → Add-ons → Add-on Store → ⋮ → Repositories** and add:
   ```
   https://github.com/steelcuts/ir2mqtt-addon
   ```
2. Find **IR2MQTT** in the store and click **Install**.
3. In the **Configuration** tab, set your MQTT broker details.
4. Click **Start** — the UI opens via the sidebar.
